# ImgPDF
A python script  🐍 that merges multiple images 📁 into one PDF file 📃.

### Run 🏁
´´´
python ImgPDF.py
´´´
###  How to use it? 🧠❔

Enter the path of the folder with images:
´´´
>path: C:\folder\with\images
´´´
Set the margin:
´´´
>margin: [num_int]
´´´
